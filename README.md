How to run application:

cd into API:
- npm install express sequelize mysql2 cors argon2 --save
- npm install apollo-server-express
- npm install express cors
- npm start

cd into frontend:
- npm install bootstrap react-star-rating-component --force
- npm start
    for testing:
    - npm install jest
    - npm test
